Quickstart

Run the minimum viable HyphaMind loop locally. These instructions assume you have the docs and automation repository checked out.

1. Prepare a virtual environment

From the repository root:

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

Always activate the environment when running smoke tests so the correct validators and policy packages load.

2. Run the Chorus smoke lane

Kick off the baseline orchestration checks:

make chorus-smoke
make chorus-receipts-check

The second target validates that every span carries the required consent attributes before the run is considered healthy.

3. Inspect REFRAIN output

REFRAIN monitors expected consent entropy. When ECE > 0.08 (EYE’s default threshold), the system writes mire_event.json alongside contextual spans.

  • MIRE event — describes the anomaly that tripped REFRAIN.
  • SCAR marker — confirms the pause is recorded with provenance.
  • Operator action — review the run, annotate, and resume intentionally.
